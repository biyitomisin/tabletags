20 HTML TAGS AND THEIR USES.

  1. !DOCTYPE: Defines the document type.
  2. html: Defines an HTML document.
  3. head: Contains metadata/information for the document.
  4. title: Defines a title for the document.
  5. body: Defines the document's body.
  6. h1 to h6: Defines HTML headings.
  7. p: Defines a paragraph.
  8. br: Inserts a single line break.
  9. hr: Defines a thematic change in the content.
  10. form: Defines an HTML form for user input.
  11. input: Defines an input control.
  12. button: Defines a clickable button.
  13. option: Defines an option in a drop-down list.
  14. output: Defines a result of a calculation.
  15. img: Defines an image.
  16. map: Defines a client-side image map.
  17. area: Defines an area inside an image map
  18. figure: Specifies self-contained content.
  19. picture: Defines a container for multiple image resources.
  20. svg: Defines a container for SVG graphics.
