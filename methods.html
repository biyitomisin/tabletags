INLINE STYLING.

Advantages

  1. Specificity: Inline styles have the highest level of specificity, which means they can easily override any other styles applied to the element.
  2. Convenience: Inline styles are convenient for making quick, specific changes to indiviual elements without affecting the rest of the document.
  3. Portability: When copying and pasting HTML elements, inline styles are carried along with the element, making it easier to maintain the styling.

Disadvantages

  1. maintainance: Inline styles can make the HTML document harder to maintain, as styles are mixed with the content and not centralized in a seperate stylesheet.
  2. Readability: Large amounts of inline styles can make the HTML code less readable and harder to understand.
  3. Reusability: Inline styles cannot be reused across multiple elements, leading to redundant code and increased file sizes.

INTERNAL STYLING.

Advantages

  1. Internal CSS has higher specificity than external CSS, which allows you to override external styles more easily within the same HTML file.
  2. Internal CSS reduces HTTP requests, enhancing performance.
  3. Internal CSS is relatively easy to implement we can use class and id selectors in this stylesheet.

Disadvantages

  1. Repetition in HTML files.
  2. Increased file size.
  3. Reduced code usability.

EXTERNAL STYLING.

Advantages

  1. With the help of External Style sheets, the styles of numerous documents can be organized from one single file.
  2. In External Style Sheets, classes can be made for use on numerous HTML element types in many forms of the site.
  3. In complex contexts, methods like selector and grouping can be implemented to apply styles.

Disadvantages

  1. An extra download is essential to import style information for each file.
  2. The execution of the file may be deferred till the external style sheet is loaded.
  3. While implementing styel sheets, we need to test Web pages with multiple browsers in order to check compatibility issues.
